# 多行业客服智能体模板矩阵可视化预览

本目录是代码开发员 Agent 为“多行业模板矩阵 + 标准商业化交付包”阶段制作的静态可视化结果。

## 打开方式

直接双击打开 `index.html` 即可预览，不需要构建、不需要安装依赖、不需要运行本地服务。

## 预览页展示内容

- 做了什么：三行业客服智能体模板矩阵、配置蓝图、验收方案和交付资产沉淀。
- 为什么这么做：让后续客户交付不再从零开始，降低 Dify 配置、FAQ 整理和验收成本。
- 目前做到什么程度：模板、文档资产、三行业 FAQ 测试集与 Dify 实装配置蓝图已完成，三行业 Dify 线上样板应用仍是下一步。
- 负责人分工：产品经理、学习研究、技术验证、Agent 搭建师、UI / 交付设计、代码开发员。
- 三行业矩阵：教育培训、家政清洁、本地门店。
- 交付资产：Spec、Dify 模板、技术验证蓝图、商业化复制包、静态预览页。
- 下一步：Dify 三行业实装、90 条测试集跑测、RAG 验收、客户演示材料整理。

## 当前边界

当前完成的是模板矩阵、三行业 FAQ 测试集、配置蓝图、验收方案和可视化预览沉淀。

三行业 Dify 线上实装、真实应用发布、线上 WebApp / embed / API 接入和真实客户验收，仍属于下一阶段。
