# 智能客服业务智能体 Dify 模板

用途：作为首个可售卖业务智能体的搭建材料包。

包含文件：

- `knowledge-base-sample.md`：样例知识库。
- `knowledge-base-ai-agent-service-sales.md`：售卖业务智能体和人工客服协同服务的通用售前知识库。
- `system-prompt.md`：系统提示词。
- `test-cases.md`：验收测试问题。
- `workflow-blueprint.json`：平台无关工作流蓝图，用于手动映射到 Dify Chatflow。

说明：当前文件不是 Dify 官方导出 DSL，避免伪造导入格式。实际 Dify 环境准备好后，按 `workflow-blueprint.json` 手动创建 Chatflow。
