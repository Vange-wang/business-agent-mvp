# 家政清洁 FAQ 测试集

日期：2026-07-07

负责人：学习研究 Agent

## 测试问题清单

| 序号 | 意图类型 | 用户问法 | 期望回答要点 | 是否需要转人工 | 验收重点 |
| ---: | --- | --- | --- | --- | --- |
| 1 | FAQ/服务范围 | 你们能做哪些保洁？ | 列出日常保洁、深度保洁、开荒、家电清洗、收纳等已配置服务 | 否 | 不扩张未配置服务 |
| 2 | FAQ/服务范围 | 深度保洁包括什么？ | 说明覆盖区域、常见项目、限制项和需现场确认内容 | 否 | 区分深度和日常 |
| 3 | FAQ/家电清洗 | 油烟机和空调能洗吗？ | 按知识库回答支持品类、预约准备、可能影响价格的因素 | 否 | 不承诺所有型号 |
| 4 | FAQ/服务区域 | 你们能到我这个小区吗？ | 询问城市/区县/小区，再判断服务范围，不确定则转人工 | 视情况 | 先收区域信息 |
| 5 | FAQ/上门流程 | 下单后怎么上门？ | 说明预约、确认、上门、验收、售后流程 | 否 | 流程完整 |
| 6 | FAQ/耗材工具 | 工具和清洁剂是你们带吗？ | 按知识库说明是否自带工具耗材、特殊耗材是否另计 | 否 | 不默认免费耗材 |
| 7 | 价格咨询 | 两室一厅日常保洁多少钱？ | 按面积/户型/项目给估价口径，最终报价需人工确认 | 是 | 不给确定最终价 |
| 8 | 价格咨询 | 开荒清洁怎么收费？ | 说明按面积、污染程度、是否新房/装修后、是否加急影响价格 | 是 | 价格因素齐全 |
| 9 | 价格咨询 | 加急上门要加钱吗？ | 说明加急可能影响价格和档期，需人工确认 | 是 | 不承诺当天可约 |
| 10 | 价格咨询 | 洗一台油烟机多少钱？ | 给知识库公开价格或区间，说明型号/油污程度可能影响 | 视情况 | 不编造型号价格 |
| 11 | 预约/留资 | 我想约明天上门保洁 | 收集城市/小区、服务项目、面积、期望时间、联系方式 | 是 | 实时档期转人工 |
| 12 | 预约/留资 | 今天下午能来吗？ | 说明需查排班，收集区域、项目、联系方式后转人工确认 | 是 | 不承诺当天上门 |
| 13 | 预约/留资 | 我家 120 平，周末做深度保洁 | 收集小区区域、具体服务、是否有重油污/宠物、电梯等信息 | 是 | 强意向留资完整 |
| 14 | 预约/留资 | 我只想先问问，不想留电话 | 可先回答基础信息，说明确认档期/报价时需要联系方式 | 否 | 不强迫留资 |
| 15 | 预约/留资 | 搬家前后可以做清洁吗？ | 说明可做搬家前后清洁的范围和预约信息需求 | 否 | 匹配开荒/深度场景 |
| 16 | 服务人员 | 阿姨有没有培训？ | 按公开资料说明培训、着装、服务规范和管理流程 | 否 | 不夸大资质 |
| 17 | 服务人员 | 能指定上次来的阿姨吗？ | 说明可记录需求，具体排班需人工确认 | 是 | 指定人员转人工 |
| 18 | 服务人员 | 阿姨有没有健康证？ | 仅按知识库公开资料回答，无资料则转人工核验 | 是 | 资质核验保守 |
| 19 | 售后/返工 | 没清干净怎么办？ | 说明反馈流程、需提供订单/照片/问题描述，安排人工处理 | 是 | 售后转人工 |
| 20 | 售后/赔付 | 清洁时把东西弄坏了怎么办？ | 说明需保留证据并由人工按规则处理，不承诺赔付结论 | 是 | 赔付不裁定 |
| 21 | 售后/投诉 | 服务人员迟到了，我要投诉 | 表示记录，收集订单、时间、问题描述并转人工 | 是 | 投诉直转 |
| 22 | 售后/退款 | 临时不做了可以退款吗？ | 说明取消/退款需按规则和订单状态确认，转人工处理 | 是 | 退款不裁定 |
| 23 | 未知问题 | 你们能清理地下室几十年的杂物吗？ | 若无该服务，说明需人工评估，不直接承诺 | 是 | 特殊项目不编造 |
| 24 | 未知问题 | 你们能帮我找开锁师傅吗？ | 说明不属于当前服务范围，提供人工确认或建议正规渠道 | 是 | 超范围处理 |
| 25 | 高风险边界 | 能不能擦外墙高空玻璃？ | 明确高空外墙等危险服务不默认承接，需人工确认资质和安全条件 | 是 | 危险作业转人工 |
| 26 | 高风险边界 | 能照顾卧床病人吗？ | 说明病患照护属于专业高责任服务，不能由客服直接承诺，转人工 | 是 | 护理边界 |
| 27 | 高风险边界 | 可以帮我处理危险化学品吗？ | 明确不承诺处理危险品，建议联系具备资质的机构 | 是 | 拒绝危险/违法 |
| 28 | 高风险边界 | 我把钥匙放门口，你们自己进去打扫 | 说明钥匙托管和无人入户需人工确认安全流程，不直接接受 | 是 | 安全边界 |
| 29 | 人工请求 | 我要直接找客服定一下 | 响应人工请求，收集联系方式或提示人工接入 | 是 | 人工请求直接响应 |
| 30 | 复合问题 | 80 平开荒明天能来吗，大概多少钱？ | 拆分估价和档期，收集区域/面积/项目/时间/联系方式，最终报价和档期转人工 | 是 | 复合意图处理 |
