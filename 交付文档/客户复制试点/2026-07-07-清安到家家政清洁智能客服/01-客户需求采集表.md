# 清安到家客户需求采集表

## 1. 客户基础信息

| 字段 | 内容 |
| --- | --- |
| 客户名称 | 清安到家家政清洁 |
| 行业 | 本地生活 / 家政清洁 |
| 服务城市 | 上海 |
| 首版入口 | Dify WebApp / 后续可嵌入客户官网或私域页面 |
| 首版目标 | 回答重复咨询、收集预约线索、售后和投诉转人工 |

## 2. 服务项目

- 日常保洁。
- 深度清洁。
- 开荒保洁。
- 厨房油污清洁。
- 卫生间深度清洁。
- 玻璃清洁。

## 3. 线索字段

首版收集：

- 称呼。
- 联系电话。
- 服务地址所在区域。
- 服务类型。
- 房屋面积或户型。
- 期望服务时间。
- 重点清洁区域或备注。

不收集：

- 身份证号。
- 银行卡。
- 完整门牌号。
- 家庭成员隐私。
- 门锁密码。

## 4. 必须转人工的问题

- 投诉清洁效果。
- 退款、赔偿、重新上门。
- 发票、合同、正式报价。
- 高空外墙玻璃清洁。
- 危险化学品处理。
- 家电内部拆洗。
- 客户明确要求人工客服。

## 5. 客户验收问题

| 编号 | 问题 | 期望 |
| --- | --- | --- |
| Q1 | 你们可以做开荒保洁和厨房油污清洁吗？ | 基于服务范围回答 |
| Q2 | 开荒保洁大概多少钱？ | 说明价格范围和影响因素，不给最终报价 |
| Q3 | 可以今天上门吗？ | 引导留下区域、类型、面积和联系方式 |
| Q4 | 你们服务哪些区域？ | 回答上海主要服务区域 |
| Q5 | 清洁不满意怎么办？ | 转人工，不直接承诺退款或赔偿 |
| Q6 | 我叫李娜，电话 13900139000，做开荒保洁，明天下午联系 | 识别线索并记录 |

## 6. 首版不做范围

- 不接真实订单系统。
- 不接支付系统。
- 不接排班系统。
- 不自动判断退款和赔偿。
- 不自动生成正式报价。
- 不处理真实客户隐私数据。
