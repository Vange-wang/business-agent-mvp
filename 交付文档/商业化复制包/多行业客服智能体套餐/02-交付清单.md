## 3. 知识库整理清单

| 模块 | 必填内容 | 完成标准 | 状态 |
| --- | --- | --- | --- |
| 基础介绍 | 客户是谁、做什么、服务谁 | 用户能快速理解业务 |  |
| 服务项目 | 项目名称、内容、适用对象 | 不遗漏主要服务 |  |
| 服务流程 | 咨询、预约、确认、交付、售后 | 用户知道下一步怎么做 |  |
| 价格边界 | 价格范围或影响因素 | 不做未经确认的最终报价 |  |
| 线索字段 | 称呼、联系方式、需求、区域等 | 字段必要且不过度收集 |  |
| 转人工规则 | 投诉、退款、合同、复杂报价等 | 高风险问题不由智能体裁定 |  |
| 兜底话术 | 知识库无答案时如何回复 | 不编造，不硬答 |  |

## 4. 上线入口清单

| 入口类型 | 适用情况 | 需要客户提供 | 状态 |
| --- | --- | --- | --- |
| Dify WebApp 链接 | 快速上线、试点验证 | 客户确认可对外发布链接 |  |
| 网站 iframe 嵌入 | 客户已有官网或落地页 | 页面位置、尺寸、入口文案 |  |
| 二维码入口 | 门店、海报、私域使用 | 二维码投放位置和说明文案 |  |
| API 接入 | 需要自定义 UI 或系统联动 | 技术对接人、接口需求、数据去向 |  |

首版建议优先使用 Dify WebApp 或 iframe 嵌入。API 接入作为二阶段增强，需单独确认范围、工期和数据安全边界。

## 5. 不含项确认

以下内容不包含在首版标准套餐内，若客户需要，应另行评估：

- 自动付款、自动退款、自动赔偿。
- 自动生成或审核合同。
- 自动承诺最终报价、最终排期或服务结果。
- CRM、企微、飞书、抖音、小红书等深度双向集成。
- 多租户后台、权限系统、自动计费系统。
- 生产隐私数据托管、数据库运维、客户内部账号长期代管。
- 医疗、法律、金融等专业结论或风险裁定。
